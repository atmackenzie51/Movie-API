myFlix Documentation Page

This goal of this API is to allow you to return a list of movies to as well as information about those movies to users.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET none none JSON object holding data of all movies
Return data of a movie by its title /movies/:title GET title: title.name none { "title": "movie title", "description": "description of movie", "genre": { "id": "value made by uuid", "name": "name of the genre", "description": "description of the genre" }, "director": { "id": "value made by uuid", "name": "name of director", "birth": "year the director was born" }, "image_Url": "URL of the movie cover" }
Return data of a specific genre /movies/genre/:genreName GET "genre": { "name": "name of the genre", } none "genre": { "id": "value made by uuid", "name": "name of the genre", "description": "description of the genre" }
Return data of a specific director /movies/directors/:directorName GET "director": { "name": "name of the director", } none "director": { "id": "value made by uuid", "name": "name of director", "birth": "year the director was born" }
Allows new users to register /users POST "name": "name of the user", "favoriteMovies": [ { "title": "title of movie" }] "user": { "name": "name of the user", "favoriteMovies": [ { "title": "title of movie" }] } "user": { "id": "value made by uuid", "name": "name of the user", "favoriteMovies": [ { "title": "title of movie" }] }
Allows users to update their name /users/:id PUT "name": "name of the user" "user": { "name": "new name of the user" } "user": { "id": "value made by uuid", "name": "new name of the user", "favoriteMovies": [ { "title": "title of movie" }] }
Allows users to add their favorite movies /users/:id/:movieTitle POST "favoriteMovies": "Name of Movie" "movieTitle": "title of the movie" "user": { "id": "value made by uuid", "name": "new name of the user", "favoriteMovies": [ { "title": "title of movie" }] }
Allows users to delete their favorite movie /users/:id/:movieTitle DELETE "favoriteMovies": "movie title to remove" "title": "movie title" Movie title was deleted from user's favorite list
Allows users to delete their profile /users/:id DELETE "id": "user id" none user was deleted
